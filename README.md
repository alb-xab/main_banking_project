# Заготовка для банковского приложения 

## Описание:

Данный проект не представляет ныне какого-либо интереса и не преследует коммерческие цели. Он создан для оттачивания навыков работы и оформления кода для молодых и неопытных будущих специалистов.

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/alb-xab/main_banking_project.git
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
## Использование:

1. Откройте папку с файлами. Файлы с функциями находятся в папке scr.
2. Для теста функций вызовите функцию внутри print, чтобы увидеть её результат.
Пример:
```
print(get_mask_card_number(1234567898765432))
```
3. Увиденные результаты оцените по шкале от 1 до 10 и оставьте свой отзыв, или свяжитесь с разработчиком, чтобы передать 
ему свои советы, пожелания или угрозы в расправе, чтобы он стал лучше работать.
4. Спасибо за внимание!

## Информация о модулях.
masks - Предназначен для маркировки переданных номеров счёта или карты.

widget - Использует функции masks для маркировки переданных ей данных, включая название карты или счёта, 
а также для переформатирования переданной ему даты. 

processing - Предназначен для фильтрации списка или же его сортировки по ключу.

generators - В ней описаны функции для работы со списками словарей, в которых можно получить фильтрацию по ключу 
валюте, а также получить информацию по выполненной операции. Также добавлена функция для генерации 
и выдачи случайного набора чисел 16-численного формата карты.


## Тестирование
1. Откройте папку с файлами тестирования. Файлы находятся в папке tests.
2. Для теста функций вызывайте каждую по отдельности, либо же все вместе с помощью pytest.
3. Результаты моих тестов вы можете найти в папке htmlcov. По моим тестам вышло 100% покрытие кода.
4. Спасибо за внимание!
